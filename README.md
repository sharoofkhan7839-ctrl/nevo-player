# Nevo Player

The Ultimate Media Player for Android - Video, Music, and URL Streaming.

## Features

- **High-Quality Playback:** Support for FHD, HD, and SD video resolutions.
- **Music Player:** Integrated audio playback.
- **Network Streaming:** Stream any video directly from a URL.
- **AMOLED Dark Theme:** Beautiful, clean pure-black dark theme.
- **Continue Watching:** Remember where you left off.
- **Folder Management:** Organize and browse your media files easily.
